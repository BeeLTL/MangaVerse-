# MangaVerse-
This project involves the development of a manga app that uses the Mangadex API to scrape data. The developer switched to Mangadex and was able to accomplish it in under four days using ChatGPT for inspiration, StackOverflow for basic tasks

## **Features**
- Browse a vast collection of manga titles
- Read manga chapters quickly and easily
- Search for any manga using the search bar

## **Getting Started**
To use Manga App, simply clone the repository to your local machine using Git. Then, navigate to the project directory and run the application using the command line. You can then browse and search for your favorite manga titles, and start reading them right away.
